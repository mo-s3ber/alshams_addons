PURCHASE ORDER EXCEL REPORTS

In this you can download purchase order excel report based on start date, end date, purchase order status and vendor and you can use "Libra Office" only.

Step 1:

When you install the module, there appears a new menu item in Purchase that is Purchase Order Excel Report.

Step 2:

When you click on Purchase Order Excel Report Menu Item a wizard will open and you can see it will take by default month start date and end date and you can select purchase order status and vendor and based on the data it will print excel report.

Step 3:

When you click Generate Report Button then Purchase Order Report file will appear and you can download it by clicking on the link.

Step 4:

When you click on the link excel sheet is downloaded and you can see the Purchase Order Excel Sheet.

Step 5:

In Purchase Order Excel Sheet you can see multiple purchase orders.

Step 6:

When you enter purchase order status or vendor for which there is no record in puchase order then this warning comes.

Step 7:

If you enter start date greater and end date smaller then this validation error comes.

Installation Guide:

Before installing this module you have to install following packages in your system.


  • sudo pip install xlrd

  • sudo pip install xlwt
